This Website

This website is designed to both a wiki type resource for various bits of knowledge I have picked up doing hobbies, as well as a blog. It's a completely static website, no JavaScript, only HTML and CSS. I write the content for the website in Markdown files which are then converted to HTML using a Golang program whenever I push to the GitHub repository.
