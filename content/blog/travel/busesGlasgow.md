# Riding All Buses - Glasgow

Glasgow has a lot of bus operators, so I need to be specific with what "Glasgow" means. To be consistent with my other cities, I'm going to define "Glasgow" as the [area highlighted when you search "Glasgow" on Google Maps](https://www.google.co.uk/maps/place/Glasgow). Of course some buses will have a few stops within this zone, so should those be included? I'm going to stick to my "coaches not included" idea, and if I think a service looks more like a coach service, I won't include it.
