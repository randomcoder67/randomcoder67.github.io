# Raspberry Pi Cyberdeck

Project to build a portable cyberdeck using a Raspberry Pi

## Goal

The goal is a somewhat portable cyberdeck, build around a Raspberry Pi, that can be used for basic Linux computing tasks (coding, taking notes, writing this website etc).  
I don't have access to a 3D printer so custom cases aren't an option.
